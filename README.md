# 🌊 My Ocean View: 교사용 도구 모음

안녕하세요! **My Ocean View**는 교실 환경에서 선생님들의 업무 효율을 높여줄 수 있는 간단한 웹 도구 모음입니다. 특히 수학 학습 자료 준비와 모둠 활동 배정에 유용합니다.

## ✨ 주요 기능

| 기능 | 설명 | 예상 사용자 |
| :--- | :--- | :--- |
| **수학 학습지/퀴즈 생성기** | 학년별, 난이도별 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈) 학습지를 자동으로 무제한 생성하고, PDF로 즉시 저장하여 인쇄할 수 있습니다. | 초등 교사, 수학 강사 |
| **모둠 편성/활동쪽 만들기** | 입력된 이름 목록을 바탕으로 팀 수나 팀당 인원수에 맞게 모둠을 랜덤으로 자동 편성하고, 그 결과를 인쇄하여 활동쪽지로 활용할 수 있습니다. | 모든 교사 |

---

## 🛠️ 사용 방법

1.  **접속:** [**My Ocean View 페이지 바로가기**](https://jing4948-lgtm.gitjub.io/jing4948/)
2.  **네비게이션:** 상단 네비게이션 바를 통해 **`홈`**, **`조편성`**, **`학습지`** 페이지로 쉽게 이동할 수 있습니다.

### 📚 학습지 생성기 사용법

1.  **설정 선택:** 학년, 문항 수, 문제 유형(덧셈/뺄셈/곱셈/나눗셈), 난이도(쉬움/보통/어려움)를 선택합니다.
2.  **문제 생성:** `문제 생성` 버튼을 누르면 즉시 학습지가 화면에 나타납니다.
3.  **저장/인쇄:** `인쇄/PDF 저장` 버튼을 누르고, 브라우저 인쇄 설정에서 대상을 **`PDF로 저장`**으로 선택하면 바로 인쇄 가능한 파일을 얻을 수 있습니다.

### 👥 모둠 편성기 사용법

1.  **이름 입력:** 좌측 텍스트 상자에 학생들의 이름을 한 줄에 한 명씩 입력하거나 CSV 형식으로 붙여넣습니다.
2.  **배정 설정:** 팀 수 또는 팀당 인원수를 설정합니다.
3.  **배정 실행:** `배정하기` 버튼을 누르면 오른쪽 영역에 무작위로 편성된 모둠 목록이 나타납니다.
4.  **저장/인쇄:** `인쇄/PDF 저장`을 통해 활동쪽지로 활용할 수 있습니다.
